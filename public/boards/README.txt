BOARD LOGOS — Teacher Training
==============================

Drop each board's logo here and it shows on the Training cards + course header.
Use these exact file names (PNG with transparent background looks best):

  ib.png     →  IB
  igcse.png  →  IGCSE
  cbse.png   →  CBSE
  icse.png   →  ICSE
  moe.png    →  MOE Singapore

Square-ish images work best (they're shown inside a white rounded tile,
object-contain so the whole logo is visible). ~200x200px is plenty.

These are official board trademarks — use logos you're licensed to display.
If a file is missing, the card falls back to a generic icon, so partial sets
are fine. After adding files, just refresh.
